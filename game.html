Морской бой

Правила игры

Морской бой — это стратегическая игра для двух игроков. Каждый игрок имеет игровое поле 10x10 клеток, на котором размещены корабли. Цель игры — потопить все корабли противника, стреляя по клеткам его поля.

Игроки по очереди кликают по клеткам, расположенным на поле противника. Побеждает игрок, который первым потопит все корабли противника.

Удачной игры!
